# Публикация на GitHub

На GitHub публикуется **только папка prototype** (без ТЗ, скринов и слайдов).

Git-репозиторий настроен **внутри папки prototype**. Подробная инструкция по первому пушу — в файле **prototype/GITHUB_PUSH.md**.

Кратко:
```bash
cd prototype
git remote add origin https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git
git push -u origin main
```
