# Как установить Node.js и npm на Mac

Node.js устанавливается вместе с **npm** (менеджер пакетов). Ниже три способа — выберите один.

---

## Способ 1: Официальный установщик (проще всего)

1. Откройте сайт: **https://nodejs.org/**
2. Скачайте версию **LTS** (рекомендуемая, зелёная кнопка).
3. Откройте скачанный файл (например, `node-v20.x.x.pkg`) и пройдите шаги установки.
4. После установки откройте **новое окно Терминала** и проверьте:
   ```bash
   node -v
   npm -v
   ```
   Должны отобразиться версии (например, `v20.10.0` и `10.2.0`).

---

## Способ 2: Через Homebrew (удобно для разработки)

Если у вас уже стоит Homebrew:

```bash
brew install node
```

Если Homebrew нет, его можно установить с сайта: **https://brew.sh/** (скопируйте команду с главной страницы и вставьте в Терминал).

После установки Node проверьте:
```bash
node -v
npm -v
```

---

## Способ 3: Через nvm (менеджер версий Node)

Удобно, если нужны разные версии Node на одном компьютере.

1. Установите nvm (в Терминале):
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
   ```
2. Закройте и снова откройте Терминал (или выполните `source ~/.zshrc`).
3. Установите Node.js LTS:
   ```bash
   nvm install --lts
   nvm use --lts
   ```
4. Проверьте:
   ```bash
   node -v
   npm -v
   ```

---
